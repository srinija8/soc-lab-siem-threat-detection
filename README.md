## Overview
This project demonstrates a hands-on Security Operations Center (SOC) lab focused on SIEM monitoring, threat detection, incident response, and threat hunting workflows.

The goal is to simulate enterprise SOC operations including:

- Security event monitoring (SIEM)
- Alert triage and investigation
- Threat detection and IOC correlation
- Incident response aligned with NIST 800-61
- MITRE ATT&CK technique mapping
